Anatomia HTML

HTML (HyperText Markup Language)

  • É uma linguagem de marcação que define a estrutura e a aprsentação do texto bruto.
  • Em HTML, o computador pode interpretar o texto bruto que é agrupado em elemento HTML.
  • Hipertexto é um texto exibido em um computador ou dispositivo que fornece acesso a outro texto por meio de links, também comhecidos como hiperlinks.

O HTML é composto de elementos. Esses elementos estruturam a página da web e definem o seu conteúdo.

Um elemento é composto por:

  • uma tag de abertura (<p>);
  • o conteúdo (<"Hello World!">)
  • uma tag de fechamento (</p>

Uma tag e o conteúdo entre ela são chamados de elemento HTML.
Existem muitas tags que podemos usar para organizar e exibir texto e outros tipos de conteúdo, como imagens.

O HTML é um conjunto de tags responsáveis pela marcação do conteúdo de uma página no navegador.
As tags são os elementos a mais que escrevemos usando a sintaxe <nome da tag>.
Diversas tags são disponibilizadas pela linguagem HTML e cada uma possui uma funcionalidade específica.

Títulos

Os títulos em HTML são semelhantes aos títulos em outros tipos de mídia. Por exemplo, em jornais, títulos grandes são normalmente usados para captar a atenção do leitor.
Outras vezes, os títulos são usados para descrever o conteúdo, como o título de um filme ou um artigo educacional.

O HTML segue um padrão semelhante. Em HTML, existem 6 títulos diferentes, ou elementos de título.
Os títulos podem ser usados para diversos fins, como títulos de seções, artigos ou outras formas de conteúdo.

Vejamos a seguinte lista:

  1. <h1>

  2. <h2>

  3. <h3>

  4. <h4>

  5. <h5>
  6. <h6>

O conteúdo de um <h1> é o maior, e o conteúdo de um <h6> é o menor.
O tamanho exato em que cada navegador mostra os títulos podem variar ligeiramente.

Os utilizadores também podem ajustar o tamanho do texto no seu browser.

Tabelas

  • Como criar tabelas
  • Como representar dados complexos em tabelas

Existem vários tipos de informações que precisam de ser exibidos em uma tabela ou mesa. Por exemplo: resultados desportivos, relatórios de ações, horários de comboios.

O que é um table ?

A tabela representa as informações num formato de grade.

Estrutura básica de tabela

<table>
O table é usado para criar uma tabela. O conteúdo da tabela são escritos
linha por linha.
<tr>
Podemos indicar o início de cada tabela usando a abertura da tag <tr>.
(O tr significa linha da tabela.)
<td>
Cada célula de uma tabela é representado usando a tag <td>.
(O td significa os dados da tabela.)

Vejamos um exemplo simples de uma tabela:

151530
456045

Cabeçalhos da tabela

O <th> é usado apenas como o td, mas o seu objetivo é representar o título, uma coluna ou uma linha.

Mesmo se uma célula não tem conteúdo (<th> </th>), devemos usar na mesma um th ou td para
representar a presença desse espaçamento/célula vazia.

Na tag podemos usar o atributo scope na tag th de forma a indicar o título de uma
coluna ou de uma linha.

Formulários

Tradicionalmente, termo forma se refere a um documento impresso que contêm espaços para nós preencher as informações.

Por quê Formas?

A forma mais conhecida na Web é provavelmente a caixa de pesquisa que fica bem no meio da página inicial go Google.
Além de permitir que os utilizadores pesquisem, as formas também permitem aos utilizadores executar outras funções online.

Veremos formas quando nos pedem para registrar em um site, quando fazemos compras online, e quando nos inscrevemos para newsletter ou listas de discussão.

Existem vários tipos de controle de formulário que vamos poder usar para receber informações de visitantes no nosso site.
Por exemplo: entrada de texto/username, entrada de senha ou área de texto.
